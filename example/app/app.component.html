Rotating Icons

Set attribute rotate or rotateBy to rotate an icon.

However, the attribute rotate only takes one of these values: 90, 180, 270.

To rotate the icon by an arbitrary angle, use the attribute rotateBy

Set attribute flip to flip, or mirror an icon horizontally, vertically or both.

FA Class <fa> attribute Rotation Amount
fa-rotate-90 rotate="90" 90°
fa-rotate-180 rotate="180" 180°
fa-rotate-270 rotate="270" 270°
fa-rotate-by/--fa-rotate-angle: Ndeg; rotateBy="N" arbitrary angle (in deg)
fa-flip-horizontal flip="horizontal" mirrors icon horizontally
fa-flip-vertical flip="vertical" mirrors icon vertically
fa-flip-both flip="both" mirrors icon vertically and horizontally

Combining Rotating + Flipping

You can both rotate and flip an icon. However, an element or a component can take ONLY one transformation.

Therefore, you’ll need to use a container, such as a <span> element, for additional transformation.

Apply one rotate to the container and another on the nested icon.

Stacking Icons

NOTE: Stacking ONLY supports up to 2 icons. To stack more than 2 icons, use SVG icons.

To stack icons, first, set the fa-stack class on the container element.

Then set the attribute stack="1" for the small icon component

Finally, set the attribute stack="2" for the large icon component.

When stacking 2 solid icons, the small icon will NOT be visible. Set the attribute inverse="true" of the small icon to make it standout from the stack.

The size of the stack can be adjusted by setting the size of the container element.

The order of the icons in the container reflects the order of stack.

Aligning Stacked and Regular Icons

You can use stacked icons right alongside single icons. Since stacked icons are styled by our framework’s included CSS to be twice the size of single icons, you’ll need to either size down the stacked icon or size up the single icons.

Power Transforms

Scaling

For sizing and scaling SVG icons, set the attributegrow or shrink to increase or decrease the size of icons.

Positioning

To move icons up, down, left, or right, set the attribute up, down, left, and right respectively with any arbitrary value, including decimals.

Note: Units are 1/16em.

Rotating & Flipping

To rotate or flip icons, use any combination of flip and rotate attributes.

Attribute fliptakes the following valueshorizontal, vertical or both

Attribute rotatetakes any arbitrary value (in degree), including decimals and negative values.

Masking Icons

Set the attributemaskPrefixandmaskName to add a masking icon as a background of current icon.

Layering, Text and Counters

Similar to stacking WebFont icons, Layering is a new way to place SVG icons and text on top of each other. Moreover, layering supports more than 2 icons (a limitation of stacking WebFont icons)

@ngx/fontawesome provides[faLayersText]directive, which is equivalent to FontAwesome fa-layers-text CSS class. The directive also performs all the transformations such as translating, scaling, rotating with values provided in the following attributes.

  • Translating: left/right/up/down
  • Scaling: shrink/grow
  • Rotating: rotate/flip

Note: Units are 1/16em.
